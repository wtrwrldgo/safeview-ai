SafeView

URL filters

Blocklist

SafeView will never run detection on URLs that contain any of these patterns. One pattern per line. Matching is a plain substring check against the full URL (hostname, pathname, and query string).

Example: youtube.com/@SomeChannel disables SafeView on every video from that channel.

Allowlist

If this list is non-empty, SafeView will only run on URLs that match one of these patterns. Leave it empty to allow every YouTube and Netflix URL by default.

Blocklist entries win over allowlist entries when both match the same URL.
